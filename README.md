# 幽骸サンズの弾幕ゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ジャンル：弾幕ゲーム
* 東方を参考にした弾幕ゲームです。

## ゲームの遊び方
* 東方の弾幕ゲームと同じ
* 主人公の残機は2個、弾幕を避けつつボスに攻撃を当てて撃破する。

## 操作方法
* 操作　　：WASDキー
* 低速移動：左Shift
* ボム発動：右Shift
* 攻撃　　：SPACEキー

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクター、ボスの描画
* ボスのHPバー

### 分担追加機能
* ボム実装 （担当：熊谷）：弾幕を全消去する「ボム」の実装
* 残機機能 （担当：河田）：残機をすべて失ったらゲームオーバー
* フェーズ1（担当：小林）：画面端からの攻撃
* フェーズ2（担当：岸）　：花形に広がる弾幕
* フェーズ3（担当：河田）：ビームを使った弾幕
